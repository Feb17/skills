# skills-merged

Collection of agent skills for OpenCode workflows.

## Skills

| Skill | Description |
|-------|-------------|
| **bosch-brand-style-guide** | Bosch corporate design for HTML/CSS — supergraphic, palette, logo SVG, typography, spacing, layout, imagery, tone, and accessibility. Use for Bosch-branded one-pagers, landing pages, design reviews, or brand compliance checks. |
| **bosch-procedure-documentation** | Create, rewrite, retrofit, or standardize Bosch internal Confluence procedure documents using the BD/ISA-CN template in PIX space. Preserves Confluence macros by working in storage format. |
| **create-jira-subtasks** | Break a Jira ticket into well-scoped sub-tasks grounded in workspace work, then create them via the Atlassian MCP with operational attributes inherited from the parent. |
| **jira-description-updater** | Update a Jira issue description into a complete DoD-style format using Jira context plus related Confluence documents. |
